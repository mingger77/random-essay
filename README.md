## 2026.06.22

但是哈，我最近几周吧，又是看[CUDA Programming Guide](https://docs.nvidia.com/cuda/cuda-programming-guide/index.html)，又是看油管博主的AI热点视频，感觉英语水平如猿猴环抱大火箭————起飞了，但自己什么都没干。

总之吧，我悟了，知识还得是从实践中学到，这在思政课上就是废话，在我的内心道场就是大道之一。

完事。

## 2026.06.26

操你妈，被一个水课大作业肘击了。

这个水课讲的是信息检索的知识，讲了很多，包括知网的使用，文件管理软件啥的，总之，这门课是教你如何在学术圈混的课。

但是吧，在这里我就得骂以中国知网的一系列学术数据库了，真的是，想找一些有价值的东西就难如登天，AI写检索词都不行，我操了，就尼玛是难为人来了。

真的，老师教的这一系列玩意，我说不好听的就他妈是一套繁琐的仪式，有用信息找不到一点，真的，操。

完事这个仪式我他妈最后也没学会，大作业还是一些不知真假的论文加上驴头不对马嘴的截图才算勉强混过去。

果然，就如我的人生感悟一样，真正内化于心的知识不是学到的。

你别看我带着好几个脏字，实际上我已经很克制了。

完事，不多说了。

## 2026.07.07

昨天考完大物了，我是凭着期中满分的底子外加三天速通的，理解算不上深刻。

但是，大物的题极为简单，大体全有思路，填空题除了少数几个考冷门知识点的题全部答上来了，保底90。

我在走出考场的那一刻，感到非常的高兴，因为我发明的速通理论经过实战的检验了。

不过，今天我看到一个学校帖子，是我同学发的，帖子大致内容如下：

我感觉自己非常迷茫，我啥都不会，考试题目却出乎意料的简单，分数能考很高，但感觉没学到什么真东西。

从这个帖子上，我看到了学校教育对于个人能力成长的莫大局限性。

我联想到C语言，我们学校的C语言是和大物在同一天考的。

C语言极为简单，我甚至没有用速通技巧，并且我只用了不足一个小时就答完了整张试卷。

我看到过能独立管理全栈项目的大专在校学生，也接触过独自搞不定大一年度项目的985学生。

可能，要想学到真正的东西，还是得靠自学。

......

果然，大学，大不了自学。

呵，我早就找好了网课，规划好了大二大三这几年要学点啥，课题组也进了。

至于绩点，真的，比想象中简单，期末抽出两三周速通即可。

完事。
